Cấu hình Firebase Realtime Database

Để leaderboard hoạt động real-time cho cả nhóm, bạn cần tạo 1 project Firebase miễn phí.
Vào console.firebase.google.com → Tạo project → Realtime Database → Start in test mode.
Rồi vào Project Settings → copy thông tin dán vào đây:

Database URL: vào Realtime Database → copy URL ở đầu trang (có thể là .firebaseio.com hoặc .asia-southeast1.firebasedatabase.app)
Môn học thường → thêm 1 lần, tự lặp mọi tuần theo đúng thứ đó.
Tiết học bù → thêm thủ công từng ngày cụ thể, không lặp tuần sau.
